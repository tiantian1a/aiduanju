# 输出 Schema（Markdown 视频提示词）

## 文件结构

```
output/
├── ep01-video-prompts.md             # 第1集视频提示词（Markdown）
├── ep02-video-prompts.md             # 第2集视频提示词（Markdown）
└── ...
```

Phase 2 的设计文件（角色/场景/道具 JSON）仍保留在各自目录中，作为提示词组装的输入来源。

---

## ep{NN}-video-prompts.md

每集一个 Markdown 文件，包含该集所有镜头的视频提示词。UTF-8 编码，可直接复制到即梦/可灵/Seedance 等 AI 视频工具使用。

### 格式规范

```markdown
# 第{N}集「{标题}」视频提示词

> 共 {M} 个镜头 ｜ 风格：{风格类型} ｜ 目标时长：{时长}

---

## 镜头 1/{M} · {时长}s

{完整视频提示词——中文自然语言叙述体，包含场景扩写、动作展开、切镜指令、对白、性别声明}

---

## 镜头 2/{M} · {时长}s

{完整视频提示词}

---

<!-- 后续镜头同理 -->
```

### 真实示例

```markdown
# 第1集「天降奇才」视频提示词

> 共 15 个镜头 ｜ 风格：电影写实 ｜ 目标时长：90s

---

## 镜头 1/15 · 5s

场景为中国现代医院普通病房，白色墙面干净整洁，输液架立在病床旁，柔和冷光洒满房间。镜头从张慧芬(女)的近景开始，她站在病床边，目光望向画面右侧，陈勇(男)的背影正缓缓走出病房门口，写实风格，电影级打光。切镜到张慧芬(女)的近景（胸部以上），她收回目光转身面向病床上的陈易，双手轻柔整理输液管，轻叹气，嘴唇微动说着："唉，陈易，你的医药费已经拖欠三天了..."，眼神略带无奈，超细节刻画手部动作，8K分辨率。最后切镜到两人同框的中景，张慧芬(女)整理好输液管后低头看着陈易，病房里安静无声，氛围忧虑而沉重，无背景音乐。陈勇是男性，张慧芬是女性

## 镜头 8/15 · 5s

场景为中国现代的世界杯决赛球场，绿茵草地在璀璨灯光下格外耀眼，背景中虚化的人山人海观众席沸腾喧嚣。镜头从林峰的中景开始，他觉醒后的神情狂热，猛地冲到球场边线，带起一阵破空呼啸声，汗水在剧烈运动中飞溅。切镜到林峰的面部特写，额头青筋暴起，眼神犀利如电死死盯着球场中央，双手快速结印，张嘴向场内大吼："赵小天！坎位七步，转身盲传！"最后切镜到球场内视角的远景，林峰的身影在场边显得极具爆发力，高对比度光影下，整个画面充满紧张激烈的决战氛围。
```

## 视频提示词推导公式

```
场景为{场景描述扩写}。
镜头从{角色名(性别)}的{景别}开始，{动作}，{风格}。
切镜到{角色名(性别)}的{景别}，{动作}，{对白："台词"}，{表情}。
最后切镜到{景别}，{氛围收束}。
{角色A}是{性别}，{角色B}是{性别}
```

### 派生规则

```
分镜图片提示词（内部使用，不输出）：
  → 角色用名字引用 → 用于生成静态分镜画面

视频提示词（.md 输出）：
  → 基于分镜图片提示词派生
  → 场景扩写 + 动作展开 + 切镜 + 对白 + 性别声明
  → 两者场景和角色完全对应
```

## 使用流程

生成 md 后的典型使用方式：

1. **加载角色参考图**（使用 Phase 2 的 characters/*.json 中的 prompt 先生成角色立绘）
2. **加载场景参考图**（使用 Phase 2 的 scenes/*.json 中的 prompt 生成场景图）
3. **逐镜头生成视频**：打开 `ep{NN}-video-prompts.md`，逐条复制提示词到 AI 视频工具（即梦/可灵/Seedance），配合分镜图片作为首帧，生成该镜头的视频片段
4. **后期合成**：将各镜头视频按剧本顺序拼接，添加音效和 BGM

## 平台适配

不同 AI 视频工具对提示词的处理方式不同，详见 [platform-adapters.md](platform-adapters.md)。

关键规则：
- **Seedance**：直接使用 md 中的完整提示词（天然兼容中文叙述体 + 切镜指令 + 性别声明）
- **可灵**：将包含多个切镜的提示词拆分为多个单镜头提示词
- **Runway Gen-3**：翻译为英文，单镜头描述
- **MiniMax**：简化为中文短描述
