# 输出模板（纯文字段落式）

以下为分镜脚本的完整输出模板。**所有输出使用纯文字段落式，不使用 markdown 表格。**

---

## 段落概览

```
【段落概览】

S1 | 00:00-00:13 | 大场面定场 | 压抑·静谧
  关键事件：异变天空下，祖父在书房书写
  转场：硬切

S2 | 00:13-00:27 | 独白抒情 | 衰老·执着
  关键事件：手颤仍继续写，画外音揭露太阳真相
  转场：匹配剪辑

...
```

---

## 详细镜头描述

每段包含：节头 → 逐镜头段落 → 节尾

### 节头模板

```
【S1】场景空间(内/外+地点+时段) · XX秒 · 类型词

人物：角色1, 角色2

起始状态：
  角色1 · 姿态[具体姿势] · 伤势[无伤/伤势] · 持有[道具(状态)] · 情绪[情绪词] · 朝向[面向谁/目光方向]
  角色2 · ...

空间布局：角色1在画面左1/3，角色2在画面右2/3；核心物件...

主线：一句话，≤15字
```

### 逐镜头模板

每个镜头为一个独立段落块，镜头之间用 `---` 分隔：

```
镜1 | 00:00-00:05 | 大全景 | 平视 | 16mm | 定镜 | 铺垫
画面：城市天际线剪影。天空呈病态暗橙色，太阳只有正常一半大小挂在地平线上。
站位：建筑群在画面下1/5，天空占上4/5
视线：—
台词：—
表演：镜头作为文明观察者，见证天空异变。靠构图传递渺小感。
微表情：—
音效：持续低频风声(呜咽感) · 远处金属结构嘎吱声
钩子：天空暗橙色 → 下一镜书房烛光暗橙(色彩匹配)

镜2 | 00:05-00:09 | 中景 | 平视 | 50mm | 斜侧 | 缓推 | 信息交代
画面：书房内景。书架上堆满书籍，烛台火焰摇曳。温度计显示-3°C。
站位：祖父在画面右2/3，桌面斜贯
视线：祖父 → 凝视 → 纸面
台词：—
表演：身体纹丝不动，只有右手手腕匀速移动。专注到近乎仪式感。
微表情：眉心微蹙 · 嘴角平直 · 目光不抬
音效：钢笔尖划过纸面的沙沙声 · 烛芯噼啪 · 风声(比外景轻)
钩子：祖父写字的动作节奏 → 引入窗外变暗的下一镜

...
```

### 节尾模板

```
---
结尾状态：
  角色1 · 姿态[..] · 伤势[..] · 持有[..] · 情绪[..] · 朝向[..]
  角色2 · ...
不变元素：光线方向/服装/关键道具/镜头基调
渐进变化：本段跨镜递进的情绪链/动作链
总时长：XX秒
```
